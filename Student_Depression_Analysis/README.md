# Student Depression Analysis

Dataset Description

The Student Depression Dataset was sourced from Kaggle and focuses on mental health conditions among students, including demographic and lifestyle variables.
It helps analyze how factors like academic pressure, sleep habits, social life, and family issues contribute to depression levels.

 Data Preparation

Cleaned using Excel (removed blanks, duplicates, formatted date/time).

Performed statistical analysis and correlation studies using Python.

Visualized data trends and relationships in Power BI dashboards.

Key Insights

Identified strong correlations between academic stress, sleep quality, and depression levels.

Gender and year of study showed notable differences in reported stress levels.

Created a correlation heatmap and interactive dashboard showing depression distribution across demographics.

Tools & Libraries

Excel for data cleaning

Python: pandas, numpy, matplotlib, seaborn

Power BI for visualization and storytelling
